Finance финансовый консультант для директоров

Что делает?
- Отвечает только на вопросы по финансам
- Даёт ответы по схеме: суть - рекомендация - действия - важное - итог
- Работает через GigaChat

Где взять ключи?
- BOT_TOKEN: создать бота у @BotFather
- GIGACHAT_AUTH_KEY: получить в Sber Studio (developers.sber.ru)
